Twitchbot

PHP based Twitch bot

View the Project on GitHub daeks/TwitchBot

  • Download ZIP File
  • View On GitHub

Setup

This is a PHP based twitch chat bot. No further installation or database is required.

$ git clone https://github.com/daeks/twitchbot
$ php -f twitchbot.php

Read README.md for further instructions.

Available Commands

Moderator can access all available commands. Access column specifies a minimum required user level.

CommandAccessDescription
!commands-Lists all commands available on github
!cmds-Lists all commands available for everybody in channel
!permitcmdspermitLists all commands available for permitted users in channel
!modcmdsmodLists all commands available for moderators in channel
!topcmd-Shows the most used command
!mods-Lists all moderators
CommandAccessDescription

!girlfriends-Displays the amount of the girlfriend queue
